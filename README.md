## Overview
Everytime a user creates a new repository, a new README is required to show and guide the users through. 
This application will help the users to create a profesisonal README.md file for their use. 
The users will be able to generate their own file after answering the prompts that are given. 

## Features
* Professional README.md file generation

## Technologies Used
1. Node.js
2. Inquirer
3. File System (fs)

## Installation
1. Clone repository:
    * git clone git@github.com:san1718/mc09-Professional_README_Generator.git
2. Install dependencies
    * npm install
3. Run the application:
    * node index.js
4. Answer the given prompts
5. Obtain final generated README.md file

## Summary
The users will be able to generate their own README.md files through the application.
