## Overview  
Creating a high-quality **README.md** file for every new repository can be time-consuming. This **Professional README.md Generator** automates the process, allowing users to generate a well-structured README by simply answering prompts in the command line.  

## Features  
✅ Generates a **professional README.md** 📄  
✅ Interactive **command-line prompts** using Inquirer ⚡  
✅ Automatically formats sections (Title, Description, Installation, Usage, License, etc.) 🔥  
✅ Saves the final **README.md** file for immediate use ✅  

## Technologies Used  
- **Node.js** (JavaScript runtime environment)  
- **Inquirer.js** (For interactive command-line prompts)  
- **File System (fs)** (To generate and save files)  

## Installation & Usage  
For local: 
### 1️⃣ Clone the Repository  
```sh
git clone git@github.com:san1718/mc09-Professional_README_Generator.git
cd mc09-Professional_README_Generator
```

### 2️⃣ Install Dependencies  
```sh
npm install
```

### 3️⃣ Run the Application  
```sh
node index.js
```

### 4️⃣ Answer the Prompts  
Follow the interactive prompts to enter project details.  

### 5️⃣ Obtain the Final README.md  
The generated **README.md** file will be created in the root directory! 🎉  

## Summary  
This tool simplifies README creation by guiding users through structured prompts and automatically formatting the final file. It ensures that each README includes all essential sections for a polished, professional presentation.
